취소
XSS 보안 개선 상세 가이드
기존 XSS 취약점
문제점:
• 사용자 입력을 HTML 필터링 없이 저장
• innerHTML로 직접 삽입하여 스크립트 실행
• 악성 코드가 다른 사용자에게 전파됨
적용된 보안 대책
서버 사이드 보안:
  • 위험한 HTML 태그 제거
  • 이벤트 핸들러 필터링
  • 특수 문자 이스케이프
  • 입력 길이 제한
클라이언트 사이드 보안:
  • 안전한 DOM 조작
  • 입력값 검증
  • CSP 헤더 적용
  • 보안 로그 기록
